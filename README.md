# 🎮 Xbox Series X — Landing Page

> Uma página web moderna, responsiva e animada inspirada no universo Xbox. Feita com HTML, CSS e JavaScript puros — sem frameworks, sem dependências pesadas.

---

## 🌐 Visão Geral

Este projeto é uma **landing page temática do Xbox Series X**, criada do zero como exercício prático de desenvolvimento frontend. Ela simula uma página de produto real — com navegação, seção de jogos, acordeão de perguntas frequentes e rodapé — tudo funcionando de forma fluida em qualquer dispositivo.

**Por que esse projeto existe?**
Para demonstrar que é possível construir interfaces modernas, bonitas e funcionais usando apenas as tecnologias fundamentais da web: HTML semântico, CSS bem estruturado e JavaScript simples. Nada além disso.

---

## ✨ Destaques

- 🖤 **Design dark com identidade visual** fiel ao universo Xbox (verde `#9BF00B` + fundo escuro)
- 📱 **Totalmente responsivo** — funciona bem em celular, tablet e desktop
- 🎯 **Acordeão interativo** nas perguntas frequentes, com animação suave
- 🖱️ **Micro-interações** no hover dos itens (controle, jogos, botões)
- ⚡ **Zero dependências de framework** — apenas HTML, CSS e JS nativos
- 🔤 **Tipografia Google Fonts** (Inter) para uma leitura limpa e moderna

## 🚀 Como Começar

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/xbox-landing-page.git
```

### 2. Acesse a pasta do projeto

```bash
cd xbox-landing-page
```

### 3. Abra no navegador

Clique duas vezes no arquivo `index.html` ou, se preferir, use uma extensão como o **Live Server** no VS Code para uma experiência com recarregamento automático.

> Nenhuma instalação adicional necessária. ✅

---

## 🧭 Como Usar

1. **Abra o projeto** no seu navegador
2. **Navegue pelo menu** para explorar as seções da página
3. **Passe o mouse** sobre o controle e os jogos para ver as animações
4. **Clique nas perguntas** da seção de FAQ para expandir as respostas
5. **Redimensione a janela** para ver o layout se adaptar em tempo real

Contribuições são bem-vindas! Se você encontrou um bug, tem uma ideia de melhoria ou quer adicionar algo novo, veja como participar:
